## ðŸ“– Tentang Proyek

Aplikasi ini adalah dashboard interaktif untuk mendeteksi apakah sebuah email termasuk kategori **SPAM** atau **HAM (Aman)**. Proyek ini membandingkan efektivitas berbagai kombinasi teknik *Natural Language Processing* (NLP) dan algoritma *Machine Learning*.

**Fitur Utama:**
* **Multi-Bahasa:** Mendukung input Bahasa Indonesia (dengan auto-translation ke Inggris).
* **Komparasi Model:** Membandingkan performa 8 skenario berbeda (TF-IDF vs BoW, Random Forest vs Logistic Regression, K-Fold vs Repeated Holdout).
* **Visualisasi Data:** Wordcloud dan analisis distribusi kelas.
* **Live Prediction:** Uji coba model secara real-time dengan antarmuka yang mudah digunakan.

**Dataset:**
Data yang digunakan bersumber dari [Kaggle - Email Spam Classification Dataset](https://www.kaggle.com/datasets/balaka18/email-spam-classification-dataset-csv), berisi 5.171 email.
